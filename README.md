# harbour-watches

A beautiful, minimalist landing page for Harbour Watches - showcasing timeless elegance and precision craftsmanship.

## Features

- Clean, minimalist design inspired by luxury watch brands
- Smooth scroll animations and interactive elements
- Fully responsive across all devices
- Product showcase with detailed modal popups
- Testimonial carousel with auto-rotation
- Animated counter statistics

## Tech Stack

- **Framework:** Next.js 16.1.1
- **Styling:** Tailwind CSS v4
- **Fonts:** Inter & Playfair Display
- **Language:** TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
